# 拿走设备功能说明

## 🎯 功能概述

在"我的借记"页面，学生可以对**已分配状态（状态3）**的借记任务执行"拿走设备"操作，表示从实验室拿走已审批通过并分配好的设备。

---

## 📋 状态流转

### 借记任务完整流程

```
1. 待审批 (Pending, 1)
   ├─ 学生创建借记任务
   └─ 等待教师审批
   
   ↓ (教师审批通过)
   
2. 已审批 (Approval, 2)
   ├─ 教师审批通过
   └─ 系统分配库存
   
   ↓ (库存分配成功)
   
3. 已分配 (Allocate, 3) ← 【可以执行"拿走设备"操作】
   ├─ 库存已分配
   ├─ 学生尚未拿走设备
   └─ ✅ 显示"拿走设备"按钮
   
   ↓ (学生拿走设备)
   
4. 进行中 (Ongoing, 4)
   ├─ 学生已拿走设备
   ├─ 设备在学生手中
   └─ 等待归还
   
   ↓ (学生归还设备)
   
8. 已归还 (Done, 8)
   └─ 任务完成
```

## 🔧 实现细节

### 1. 后端接口

**接口路径**：`/apps/bms/borrow/task_borrow`

**请求参数**：
```json
{
  "borrow_id": "借记任务ID",
  "code_list": ["设备编码1", "设备编码2"]  // 可选，传空数组
}
```

**响应**：
```json
{
  "retcode": 0,
  "message": "success",
  "data": {}
}
```

**后端逻辑**：
1. 验证借记任务状态为"已分配"（状态3）
2. 将借记任务状态更新为"进行中"（状态4）
3. 更新库存状态：
   - `borrowed_qty` 增加（借出数量）
   - `allocated_qty` 减少（分配数量）
   - `on_hand_qty` 减少（在手数量）
   - `avail_qty` 不变（可用数量）
4. 记录操作日志

### 2. 前端实现

#### API 函数

**文件**：`frontend/src/api/borrow.js`

```javascript
// 拿走设备（学生从实验室拿走已分配的设备）
export const takeBorrow = (data) => {
  return request({
    url: '/apps/bms/borrow/task_borrow',
    method: 'POST',
    data,
  });
};
```

#### 页面组件

**文件**：`frontend/src/pages/Borrow/MyTasks.jsx`

**关键代码**：

```javascript
// 拿走设备的处理函数
const handleTakeBorrow = async (record) => {
  Modal.confirm({
    title: '📦 确认拿走设备',
    content: (
      <div style={{ padding: '16px 0' }}>
        <div style={{ 
          background: '#e6f7ff', 
          border: '1px solid #91d5ff',
          borderRadius: 8,
          padding: 16,
          marginBottom: 16
        }}>
          <p style={{ margin: 0, color: '#1890ff', fontWeight: 600 }}>
            确认从实验室拿走设备后，状态将变更为"进行中"
          </p>
        </div>
        <p>任务ID：{record.task_id}</p>
        <p>设备：{record.equip_id}</p>
        <p>实验室：{record.lab_id}</p>
        <p>数量：{record.borrow_qty}</p>
        <div style={{ marginTop: 16, padding: 12, background: '#fffbe6' }}>
          <p>💡 提示：拿走设备后请妥善保管，使用完毕后及时归还</p>
        </div>
      </div>
    ),
    okText: '确认拿走',
    cancelText: '取消',
    okButtonProps: { type: 'primary' },
    onOk: async () => {
      try {
        await takeBorrow({ 
          borrow_id: record.task_id,
          code_list: []
        });
        message.success('✅ 拿走成功，请妥善保管设备');
        loadData();
      } catch (error) {
        message.error('❌ 操作失败');
      }
    },
  });
};

// 表格列定义中的操作列
{
  title: '操作',
  key: 'action',
  width: 150,
  fixed: 'right',
  render: (_, record) => (
    <Space>
      {record.task_status === 1 && ( // 待审批状态可以取消
        <Button 
          type="link" 
          danger 
          icon={<CloseOutlined />}
          onClick={() => handleCancel(record)}
        >
          取消
        </Button>
      )}
      {record.task_status === 3 && ( // 已分配状态可以拿走设备
        <Button 
          type="primary"
          size="small"
          icon={<CheckCircleOutlined />}
          onClick={() => handleTakeBorrow(record)}
        >
          拿走设备
        </Button>
      )}
    </Space>
  ),
}
```

## 💡 使用提示

### 给学生的提示

1. **拿走时机**：只有在任务状态为"已分配"时才能拿走设备
2. **设备保管**：拿走设备后请妥善保管，避免损坏或丢失
3. **及时归还**：使用完毕后请及时归还设备，方便其他同学使用
4. **状态查看**：拿走后任务状态会变为"进行中"，可以在列表中查看

### 给教师的提示

1. **监控管理**：可以在"所有借记"页面查看哪些设备被拿走（进行中状态）
2. **库存追踪**：通过"三级账查询"可以查看设备借出记录
3. **提醒归还**：可以主动提醒学生归还设备
