## 🔧 配置文件说明

### 需要修改的配置文件

#### 1. server/_config/conf.yaml
**必须修改项**：
```yaml
datasource:
  default:
    groups:
      - masterDsn: "root:你的密码@(127.0.0.1:3306)/bms_basic_db"
  inv:
    groups:
      - masterDsn: "root:你的密码@(127.0.0.1:3306)/bms_inv_db"
  bms:
    groups:
      - masterDsn: "root:你的密码@(127.0.0.1:3306)/bms_task_db"
```

**可选修改项**：
- 端口配置
- Kafka 配置（可删除）
- ES 配置（可删除）
- Redis 配置（可删除）

#### 2. frontend/package.json
**可选修改项**：
```json
{
  "proxy": "http://localhost:8080"
}
```

#### 3. nginx/www.bms.com.conf
**生产环境需修改**：
```nginx
# 静态文件路径
root /path/to/LabEquip-BMS/frontend/build;

# 日志路径
access_log /var/log/nginx/bms.access.log;
error_log /var/log/nginx/bms.error.log;
```
