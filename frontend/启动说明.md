# 前端启动说明

## 环境要求

- Node.js 14.0 或更高版本
- npm 或 yarn

## 安装步骤

### 1. 进入前端目录

```bash
cd frontend
```

### 2. 安装依赖

```bash
npm install
```

或使用 yarn：

```bash
yarn install
```

### 3. 启动开发服务器

```bash
npm start
```

或使用 yarn：

```bash
yarn start
```

应用将在 http://localhost:3000 启动，浏览器会自动打开。

### 4. 构建生产版本

```bash
npm run build
```

生产版本的文件会生成在 `build` 目录。

## 配置说明

### API 代理

开发环境下，API 请求会代理到 `http://localhost:8080`。

如需修改后端地址，编辑 `package.json` 中的 `proxy` 字段：

```json
{
  "proxy": "http://localhost:8080"
}
```

### 环境变量

可以创建 `.env` 文件设置环境变量：

```env
REACT_APP_API_URL=http://localhost:8080
```

## 默认账户

系统注册的用户默认为学生角色。

如需创建教师或管理员账户，需要通过后端直接插入数据库或提供管理员创建接口。

## 功能模块

### 1. 登录/注册
- URL: `/login`
- 功能：用户登录和注册

### 2. 首页
- URL: `/`
- 功能：显示统计数据和最近任务

### 3. 用户管理（教师及以上）
- URL: `/users`
- 功能：查看所有用户

### 4. 设备管理
- URL: `/equipment`
- 功能：查看设备列表，管理员可创建设备

### 5. 库存管理
- 库存查询: `/inventory/list`
- 库存操作: `/inventory/manage`（教师及以上）
- 库存任务: `/inventory/tasks`
- 三级账: `/inventory/transactions`

### 6. 借记管理
- 我的借记: `/borrow/my-tasks`
- 所有借记: `/borrow/all-tasks`（教师及以上）
- 审批管理: `/borrow/approve`（教师及以上）

## 权限说明

- **超级管理员（SuperAdmin）**：所有权限
- **管理员（Admin）**：用户管理、设备管理、库存管理、审批借记
- **教师（Teacher）**：用户管理、设备管理、库存管理、审批借记
- **学生（Student）**：查看设备、查看库存、创建和管理自己的借记任务

## 常见问题

### 1. 安装依赖失败

尝试清除缓存后重新安装：

```bash
rm -rf node_modules package-lock.json
npm install
```

### 2. 端口被占用

如果 3000 端口被占用，可以使用其他端口：

```bash
PORT=3001 npm start
```

### 3. API 请求失败

确保后端服务已启动，并运行在 `http://localhost:8080`。

检查浏览器控制台的网络请求，确认代理配置正确。

### 4. 登录后页面白屏

检查浏览器控制台是否有错误信息。

清除浏览器 localStorage：

```javascript
localStorage.clear()
```

然后重新登录。

## 技术栈

- **React 18**: UI 框架
- **React Router v6**: 路由管理
- **Ant Design 5**: UI 组件库
- **MobX 6**: 状态管理
- **Axios**: HTTP 请求
- **Day.js**: 日期处理
