## 🎨 设计系统

### 颜色方案

#### 主题色
```css
主渐变色: linear-gradient(135deg, #667eea 0%, #764ba2 100%)
主色调: #667eea (紫蓝色)
辅助色: #764ba2 (紫色)
```

#### 功能色
```css
成功色: #52c41a
警告色: #faad14
错误色: #ff4d4f
信息色: #1890ff
```

#### 中性色
```css
背景色: #f5f5f5
卡片背景: #ffffff
文字主色: #333333
文字辅色: #666666
文字灰色: #999999
边框色: #f0f0f0
```

### 字体规范

```css
页面标题: 24-28px, 粗体 (700)
卡片标题: 16-18px, 中粗 (600)
正文: 14-16px, 常规 (400)
辅助文字: 12-14px, 常规 (400)
```

### 圆角规范

```css
小圆角: 6px (按钮、输入框、Tag)
中圆角: 8px (表格)
大圆角: 12px (卡片、Modal)
超大圆角: 16px (登录卡片)
```

### 阴影规范

```css
浅阴影: 0 2px 8px rgba(0, 0, 0, 0.06)
中阴影: 0 4px 12px rgba(0, 0, 0, 0.12)
深阴影: 0 8px 24px rgba(0, 0, 0, 0.15)
主题色阴影: 0 2px 8px rgba(102, 126, 234, 0.3)
```

### 动画规范

```css
标准过渡: all 0.3s ease
淡入动画: fadeIn 0.5-0.6s ease-out
上浮效果: transform: translateY(-2px to -4px)
```

## 🚀 使用建议

### 引入通用样式
所有新页面都应该使用通用样式类：

```jsx
import React from 'react';

const MyPage = () => {
  return (
    <div className="page-container">
      <div className="page-header">
        <h1 className="page-title">页面标题</h1>
        <p className="page-description">页面描述</p>
      </div>
      
      {/* 页面内容 */}
    </div>
  );
};
```

### 使用统计卡片
```jsx
<Card className="stat-card" hoverable>
  <Statistic
    title="标题"
    value={100}
    prefix={<IconComponent />}
    valueStyle={{ color: '#667eea' }}
  />
</Card>
```

### 使用渐变按钮
```jsx
<Button className="gradient-button" size="large">
  操作按钮
</Button>
```
