# 设备图片轮播和详情页功能说明

## 📋 功能概述

为设备管理模块添加了图片轮播展示和详情页功能，用户可以在设备列表中直接查看设备图片轮播，并点击进入详情页查看所有图片和完整信息。

---

## ✨ 新增功能

### 1. **图片轮播组件 (ImageCarousel)**

**文件位置**：`frontend/src/components/ImageCarousel.jsx`

**功能特性**：
- ✅ 支持多张图片自动轮播
- ✅ 左右箭头控制切换
- ✅ 显示当前图片索引（如：1/5）
- ✅ 鼠标悬停显示控制按钮
- ✅ 点击图片可预览大图
- ✅ 无图片时显示占位符
- ✅ 响应式设计，适配不同尺寸

**使用示例**：

```jsx
import ImageCarousel from '../../components/ImageCarousel';

<ImageCarousel 
  images={['url1', 'url2', 'url3']} 
  width={200} 
  height={120}
  showControls={true}
/>
```

**参数说明**：
- `images`: 图片 URL 数组
- `width`: 轮播图宽度（默认 200px）
- `height`: 轮播图高度（默认 150px）
- `showControls`: 是否显示控制按钮（默认 true）

## 🔧 技术实现

### 1. **路由配置**


### 2. **图片轮播实现**

使用 Ant Design 的 `Carousel` 组件：

```jsx
<Carousel 
  ref={carouselRef}
  dots={false}
  afterChange={handleAfterChange}
  autoplay={false}
>
  {images.map((url, index) => (
    <div key={index}>
      <Image src={url} width={width} height={height} />
    </div>
  ))}
</Carousel>
```

**自定义控制按钮**：
- 左右箭头按钮（悬停显示）
- 图片索引指示器（底部居中）

### 3. **详情页数据加载**

```jsx
const { equipId } = useParams();  // 获取路由参数

const loadEquipDetail = async () => {
  const res = await searchEquip({});
  const equip = res.data?.list?.find(item => item.equip_id === equipId);
  setEquipData(equip);
};
```

## 🐛 修复的问题

### 问题 1：字段名不一致

**问题描述**：
- 后端 protobuf 定义使用 `image_list`
- 生成的 Go 代码 JSON tag 是 `images`
- 前端使用 `image_list` 导致数据无法正确获取

**解决方案**：
1. 修改 protobuf 定义，统一使用 `images`：
   ```protobuf
   message EquipInfo {
     // ...
     repeated string images = 10;  // 改为 images
   }
   ```

2. 重新生成 protobuf 代码：
   ```bash
   cd api_idl && bash pbgenerator/generate.sh
   ```

3. 前端统一使用 `images` 字段

### 问题 2：上传接口缺少 file_name 字段

**已在之前修复**：
- `ImageUpload.jsx` 中添加 `formData.append('file_name', file.name)`
- 确保后端能正确获取文件扩展名

## 🚀 使用流程

### 用户操作流程

1. **进入设备列表页** (`/equipment`)
   - 查看设备列表
   - 每个设备显示图片轮播

2. **查看图片轮播**
   - 鼠标悬停显示左右箭头
   - 点击箭头切换图片
   - 查看当前图片索引

3. **进入详情页**
   - 点击图片轮播 → 跳转详情页
   - 点击"详情"按钮 → 跳转详情页

4. **详情页操作**
   - 查看大图轮播
   - 点击缩略图预览
   - 查看完整设备信息
   - 点击"返回列表"回到列表页

## 🔍 注意事项

1. **图片 URL 格式**
   - 后端返回的是完整 URL（包含域名）
   - 格式：`http://domain.com/uploads/20260113/uuid.jpg`

2. **路由参数**
   - 详情页使用 `equipId` 作为路由参数
   - 通过 `useParams()` 获取

3. **权限控制**
   - 所有用户可查看详情
   - 仅管理员可编辑设备

4. **响应式设计**
   - 移动端自动调整轮播图尺寸
   - 缩略图在小屏幕上自动换行
