## 🚀 核心功能实现

### 1. 日志文件输出

**实现方式**：
- 使用 Go 标准库 `log` 包
- 自定义 `BMSLogger` 结构体
- 支持 `io.MultiWriter` 同时输出到文件和控制台

**代码示例**：
```go
type BMSLogger struct {
    logDir         string
    currentLogFile *os.File
    logger         *log.Logger
    level          LogLevel
    enableConsole  bool
    mu             sync.RWMutex
    currentDate    string
}
```

### 2. 按天切割日志

**实现方式**：
- 每分钟检查当前日期
- 日期变化时自动切换到新文件
- 文件命名格式：`bms-YYYY-MM-DD.log`

**代码示例**：
```go
func (l *BMSLogger) watchLogRotate() {
    ticker := time.NewTicker(1 * time.Minute)
    defer ticker.Stop()
    
    for range ticker.C {
        now := time.Now()
        dateStr := now.Format("2006-01-02")
        
        if l.currentDate != dateStr {
            l.rotateLogFile()
        }
        
        l.cleanOldLogs()
    }
}
```

### 3. 自动清理日志

**实现方式**：
- 保留最近 7 天的日志
- 每次切割时检查并删除过期日志
- 基于文件修改时间判断

**代码示例**：
```go
func (l *BMSLogger) cleanOldLogs() {
    cutoffDate := time.Now().AddDate(0, 0, -7)
    
    for _, file := range files {
        if info.ModTime().Before(cutoffDate) {
            os.Remove(logPath)
        }
    }
}
```

### 4. 多级别日志

**支持级别**：
- DEBUG：调试信息
- INFO：一般信息
- WARN：警告信息
- ERROR：错误信息
- FATAL：致命错误（会退出程序）

**使用示例**：
```go
log.Debugf("Debug info: %v\n", data)
log.Infof("Server started on port %d\n", 8080)
log.Warnf("Warning: %s\n", msg)
log.Errorf("Error: %v\n", err)
log.Fatalf("Fatal: %v\n", err)
```

### 5. 控制台同步输出

**实现方式**：
- 配置 `enableConsole: true` 时同时输出
- 使用 `io.MultiWriter` 实现
- 开发环境推荐开启，生产环境推荐关闭

**配置示例**：
```yaml
log:
  enableConsole: true  # 开发环境
  # enableConsole: false  # 生产环境
```

```go
func main() {
    fmt.Println(path)
    fmt.Println("configPath: " + configPath)
    // ...
    fmt.Println("Initialize success")
    s.Run()

```

**问题**：
- 使用 `fmt.Println` 输出
- 没有日志文件记录

### 优化后（main.go）

```go
func main() {
    // 加载配置
    config.DoInitWcConfigWithPath(configPath)
    
    // 初始化日志
    initialize.InitLogger()
    
    // 使用日志系统
    log.Infof("=================================================\n")
    log.Infof("LabEquip-BMS Backend Service Starting...\n")
    log.Infof("=================================================\n")
    
    // 启动服务
    go func() {
        s.Run()
    }()
    
    // 优雅退出
    quit := make(chan os.Signal, 1)
    signal.Notify(quit, syscall.SIGINT, syscall.SIGTERM)
    <-quit
    
    log.Close()
}
```

- ✅ 使用日志系统记录
