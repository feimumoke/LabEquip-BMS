# 批量文件信息接口说明

## 📋 功能概述

实现了一个批量获取文件信息的接口，用于验证文件是否存在、获取文件大小、MIME 类型等元信息。

---

## 🎯 接口信息

### 接口路径

**POST** `/apps/common/batch_file_info`

### 功能说明

- ✅ 批量查询文件信息
- ✅ 验证文件是否存在
- ✅ 获取文件大小
- ✅ 获取 MIME 类型
- ✅ 获取文件名
- ✅ 获取修改时间
- ✅ 最多支持 100 个文件的批量查询

## 🔄 实现逻辑

### 后端实现

**文件**：`apps/common/view/batch_file_handler.go`

**核心逻辑**：

```go
func (h *BatchFileHandler) BatchFileInfoView(ctx context.Context, header *web.Header, request interface{}) (interface{}, *bmserror.BMSError) {
	req := request.(*pbcommon.BatchFileInfoRequest)
	
	// 1. 校验参数
	if len(req.GetFileUrls()) == 0 {
		return nil, bmserror.NewError(constant.ErrParam, "file_urls is empty")
	}
	
	// 2. 限制批量查询数量（最多 100 个）
	if len(req.GetFileUrls()) > 100 {
		return nil, bmserror.NewError(constant.ErrParam, "file_urls count exceeds limit (max 100)")
	}
	
	// 3. 批量获取文件信息
	fileInfos := make([]*pbcommon.FileInfo, 0, len(req.GetFileUrls()))
	for _, fileUrl := range req.GetFileUrls() {
		fileInfo := h.getFileInfo(fileUrl)
		fileInfos = append(fileInfos, fileInfo)
	}
	
	return &pbcommon.BatchFileInfoResponse{
		Files: fileInfos,
	}, nil
}
```

**文件信息获取**：

```go
func (h *BatchFileHandler) getFileInfo(fileUrl string) *pbcommon.FileInfo {
	// 1. URL 转本地路径
	// /uploads/20240113/uuid.jpg -> ./uploads/20240113/uuid.jpg
	filePath := h.urlToFilePath(fileUrl)
	
	// 2. 检查文件是否存在
	fileInfo, err := os.Stat(filePath)
	if err != nil {
		// 文件不存在
		return &pbcommon.FileInfo{
			Url:    convert.String(fileUrl),
			Exists: convert.Bool(false),
		}
	}
	
	// 3. 获取文件信息
	return &pbcommon.FileInfo{
		Url:          convert.String(fileUrl),
		Exists:       convert.Bool(true),
		Size:         convert.Int64(fileInfo.Size()),
		MimeType:     convert.String(h.getMimeType(filePath)),
		FileName:     convert.String(filepath.Base(filePath)),
		ModifiedTime: convert.Int64(fileInfo.ModTime().Unix()),
	}
}
```

**MIME 类型检测**：

```go
func (h *BatchFileHandler) getMimeType(filePath string) string {
	// 1. 先根据文件扩展名判断
	ext := filepath.Ext(filePath)
	mimeType := mime.TypeByExtension(ext)
	if mimeType != "" {
		return mimeType
	}
	
	// 2. 如果无法根据扩展名判断，读取文件内容判断
	file, err := os.Open(filePath)
	if err != nil {
		return "application/octet-stream"
	}
	defer file.Close()
	
	// 3. 读取文件头部（最多 512 字节）
	buffer := make([]byte, 512)
	n, _ := file.Read(buffer)
	
	// 4. 使用 http.DetectContentType 检测
	return http.DetectContentType(buffer[:n])
}
```

## 🧪 使用场景

### 场景 1：验证上传的文件是否存在

**前端代码**：
```javascript
import { batchFileInfo } from '../api/file';

// 验证设备图片是否都存在
const validateImages = async (imageUrls) => {
  try {
    const res = await batchFileInfo({
      file_urls: imageUrls
    });
    
    if (res.retcode === 0) {
      const missingFiles = res.data.files.filter(f => !f.exists);
      if (missingFiles.length > 0) {
        console.warn('以下文件不存在：', missingFiles.map(f => f.url));
        message.warning(`有 ${missingFiles.length} 个文件不存在`);
      } else {
        message.success('所有文件都存在');
      }
    }
  } catch (error) {
    message.error('验证失败');
  }
};

// 使用
validateImages([
  '/uploads/20240113/uuid1.jpg',
  '/uploads/20240113/uuid2.png'
]);
```

## ⚠️ 注意事项

### 1. 性能考虑

- **批量限制**：最多支持 100 个文件的批量查询
- **I/O 性能**：大量文件查询会有磁盘 I/O 开销
- **建议**：合理控制批量大小，避免一次查询过多文件

---

### 2. 安全性

- **路径遍历**：接口会将 URL 转换为本地路径，需要防止路径遍历攻击
- **当前实现**：只允许访问 `./uploads/` 目录下的文件
- **建议**：添加路径校验，确保不会访问系统其他目录

**安全增强示例**：
```go
func (h *BatchFileHandler) urlToFilePath(fileUrl string) string {
	// 去除 URL 前缀
	path := strings.TrimPrefix(fileUrl, "/")
	
	// 只允许 uploads 目录
	if !strings.HasPrefix(path, "uploads/") {
		return ""  // 非法路径
	}
	
	// 防止路径遍历（.. 或 .）
	if strings.Contains(path, "..") || strings.Contains(path, "./") {
		return ""  // 非法路径
	}
	
	return "./" + path
}
```

---

### 3. 错误处理

- **文件不存在**：不会返回错误，而是 `exists: false`
- **权限问题**：如果无法读取文件，会返回 `exists: false`
- **路径格式错误**：不会报错，直接返回 `exists: false`

func (h *BatchFileHandler) BatchFileInfoView(...) {
	// 使用 goroutine pool 并发查询
	results := make(chan *pbcommon.FileInfo, len(fileUrls))
	
	for _, url := range fileUrls {
		go func(fileUrl string) {
			results <- h.getFileInfo(fileUrl)
		}(url)
	}
	
	// 收集结果
	for i := 0; i < len(fileUrls); i++ {
		fileInfos = append(fileInfos, <-results)
	}
